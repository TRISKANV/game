🛡️ Project: Survivors-Open-Source 
Un clon de supervivencia contra hordas (Bullet Heaven) desarrollado en Godot 4.x para Android. El objetivo es crear un motor sólido, modular y fácil de expandir con nuevos personajes y armas.

🚀 Características del Proyecto
Plataforma: Android (Optimizado para pantallas táctiles).

Motor: Godot Engine 4.x.

Lenguaje: GDScript (similar a Python, fácil de aprender).

Licencia: MIT (Abierto para todos).
